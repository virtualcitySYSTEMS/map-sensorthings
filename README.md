# @vcmap/sensorthings

> Part of the [VC Map Project](https://github.com/virtualcitySYSTEMS/map-ui)

## Description

**sensorthings** is a plugin for the VC Map that makes it possible to display sensor locations and the corresponding data from the [SensorThings API](https://www.ogc.org/publications/standard/sensorthings/) as a VectorTiles Layer.
